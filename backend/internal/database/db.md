# Регистрация
Пользователь вводит логин
1) Генерация ключей
2) Отправка их на бэкенд
3) Получение jwt токена
# Авторизация
Пользователь нажимает кнопку
1) Первый запрос на бэкенд с логином и открытым ключом
2) Получение испытания
3) Подпись испытания
4) Получение jwt токена
